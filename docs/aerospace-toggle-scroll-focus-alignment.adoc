// tag::synopsis[]
aerospace toggle-scroll-focus-alignment [-h|--help]

// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
{manpurpose}

Toggles between two alignment modes for the focused column in a scrolling layout:

- `smart` -- The viewport stays still when the focused column is already visible; scrolls to the left edge only when the column is off-screen (default)
- `center` -- The focused column is always centered in the viewport
